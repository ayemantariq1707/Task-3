# Language Learning App
A user-friendly application designed to help users learn multiple languages through words, phrases, and sentences. 
The app supports five languages and provides an interactive way to improve vocabulary and basic communication skills.

---
## Features

- Learn 5 different languages:
  - English
  - Urdu
  - German
  - French
  - Turkish
- Vocabulary learning (words, phrases, sentences)
- Easy navigation between languages
- Simple and intuitive user interface
-  Beginner-friendly learning experience
---
## 🛠️ Tech Stack
- Android Studio 
- Kotlin
- XML 
---
##  How It Works

- Users select a language from the available options
- The app displays words, phrases, and sentences for learning
- Users can navigate through different categories
---
##  Purpose of Project
- Practice multi-language content handling
- Improve UI/UX design skills
